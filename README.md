# Pitch Skills

The open-source skill tree for founders who pitch in the arena.

This repository is architected independently, inspired by structured skill-library formats,
but fully rebuilt for pitching, fundraising, and operator-level execution.

## Categories

- fundamentals
- structure
- strategy
- fundraising
- delivery

Each skill lives in:

skills/<category>/<skill-name>/SKILL.md

## Start Here

- Browse all skills: `skills/README.md`
- Contribute a skill: `CONTRIBUTING.md`
